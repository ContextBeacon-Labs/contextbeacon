## Why this repository qualifies

ContextBeacon is a public, actively maintained open-source tool for coding agents.
It helps maintainers scan a repository, generate agent-ready context files, detect
stale instructions, and keep those files synchronized with the source tree.

The repository now has:

- a public GitHub organization and repository
- a custom organization avatar
- a working Node-based CLI with `scan`, `generate`, `lint`, and `sync`
- generated `AGENTS.md` and `CLAUDE.md` files
- CI that checks formatting, types, and tests

## How API credits will be used

API credits will be used to improve the repository intelligence workflow itself:

- repository scanning heuristics
- richer generated context files
- stale-path detection
- CLI and automation polish
- validation tests for repo shape changes
